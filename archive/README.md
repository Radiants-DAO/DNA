# Archive

Historical and implemented documentation lives here.

- `plans/` stores completed or superseded implementation plans.
- `brainstorms/` stores rationale for work that already shipped or was folded into later docs.
- `conversion/` and `prompts/` store legacy DNA conversion material.
- `production-readiness/`, `reports/`, `qa/`, and `research/` store dated audits and loop outputs.
- `ops/` stores dated execution trackers, session logs, and retired workflow notes.
- `rad-os/` stores legacy app-local docs that were consolidated out of `apps/rad-os/docs/`.
