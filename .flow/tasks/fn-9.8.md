# fn-9.8 Accumulator UI

## Description
Build the UI for viewing accumulated edits.

**Location:** Left panel overlay/tab (can split with Layers)

**Features:**
- Show pending edits grouped smartly (by file, by type)
- Edit count badge on tab/header
- Clear all button
- Individual edit removal
- Optional: user annotations for intent

**Design:**
- Compact list view showing property: old → new
- File grouping headers when multiple edits in same file
- Visual distinction between edit types (token, typography, etc.)
