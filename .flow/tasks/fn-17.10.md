# fn-17.10 Create per-folder MOC index files

## Description
Create per-folder MOC (Map of Content) index files.

## MOC Files to Create

- `~/Desktop/vault/radflow/00-Index/index.md` - Main vault index
- `~/Desktop/vault/radflow/01-Architecture/index.md` - Architecture docs
- `~/Desktop/vault/radflow/02-Features/index.md` - Domain docs
- `~/Desktop/vault/radflow/03-Sprints/index.md` - Sprint/progress docs
- `~/Desktop/vault/radflow/04-References/index.md` - Reference materials
- `~/Desktop/vault/radflow/99-Archive/index.md` - Archived content

## Template

```yaml
---

date: 2026-01-17

- [[doc-2]] - Description
```

## Main Index (00-Index)

## Quick Commands

```bash
# Create MOC files
for dir in 00-Index 01-Architecture 02-Features 03-Sprints 04-References 99-Archive; do
  cat > ~/Desktop/vault/radflow/$dir/index.md
done
```
