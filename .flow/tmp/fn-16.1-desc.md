Create the vault folder structure at `~/Desktop/vault/radflow/` with all required directories and initialize Obsidian configuration.

## Folder Structure

```
~/Desktop/vault/radflow/
├── .obsidian/           # Obsidian config
├── 00-Index/            # MOC files
├── 01-Architecture/     # System overview
├── 02-Features/         # 6 domain docs
├── 03-Sprints/          # Progress tracking
├── 04-References/       # Rust patterns, research
│   └── research/
└── 99-Archive/          # Flow backup destination
```

## .obsidian Setup

Create minimal Obsidian config:
- `app.json` with basic settings
- Empty `plugins/` directory for future Claudesidian install

## Commands

```bash
# Create structure
mkdir -p ~/Desktop/vault/radflow/{.obsidian/plugins,00-Index,01-Architecture,02-Features,03-Sprints,04-References/research,99-Archive}

# Create minimal app.json
echo '{"promptDelete": false, "alwaysUpdateLinks": true}' > ~/Desktop/vault/radflow/.obsidian/app.json

# Verify
ls -la ~/Desktop/vault/radflow/
```
