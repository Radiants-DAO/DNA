## Task Breakdown

### Phase 1: Git Commands (P0)

**fn-11.1: Create Git Module** (2-3h)
- Create `src-tauri/src/commands/git.rs`
- Add to `mod.rs`
- Register commands with Tauri
- Use git CLI (not git2 per CLAUDE.md)

**fn-11.2: Implement git_status Command** (2h)
- Run `git status --porcelain`
- Parse output to structured data
- Return modified/staged/untracked files
- Handle non-git directories

**fn-11.3: Implement git_diff Command** (2h)
- Run `git diff` and `git diff --staged`
- Return diff content
- Support file-specific diffs
- Handle binary files

**fn-11.4: Implement git_commit Command** (3-4h)
- Run `git add -A && git commit -m "message"`
- Support optional file selection
- Return commit hash
- Handle commit hooks

### Phase 2: Snapshot System (P0)

**fn-11.5: Implement Snapshot Create** (3-4h)
- Create named snapshots (stash-like)
- Store current state before changes
- Include commit hash reference
- Limit snapshot count

**fn-11.6: Implement Snapshot Restore** (2-3h)
- Restore to previous snapshot
- Discard current changes
- Confirm before destructive action
- Remove restored snapshot

### Phase 3: UI Integration (P0)

**fn-11.7: Create Commit Dialog** (3-4h)
- Create `CommitDialog.tsx`
- Show changed files
- Commit message input
- Wire to Cmd+S shortcut

**fn-11.8: Implement Undo Everything** (2-3h)
- Add "Discard Changes" option
- Restore to last commit/snapshot

## Technical Notes

### Git CLI Wrapper
```rust
// src-tauri/src/commands/git.rs
use std::process::Command;

#[tauri::command]
#[specta::specta]
pub fn git_status(project_path: PathBuf) -> Result<GitStatus, String> {
    let output = Command::new("git")
        .args(["status", "--porcelain"])
        .current_dir(&project_path)
        .output()
        .map_err(|e| e.to_string())?;

    // Parse output...
}
```

### Why git CLI vs git2
Per CLAUDE.md: "git2 → git CLI". The CLI approach:
- Works with user's git config
- Supports all git features
- Handles credentials properly
- Simpler error handling

### Snapshot Implementation
Store snapshots as:
```json
{
  "name": "before-typography-changes",
  "commit": "abc123",
  "created_at": "2024-01-15T10:30:00Z",
  "modified_files": ["tokens.css", "typography.css"]
}
```
