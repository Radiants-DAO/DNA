# fn-15: Assets Manager

## Overview

Implement the Assets Manager backend infrastructure. Currently the Assets panel uses mock data; this epic adds the Rust types and commands to work with real filesystem assets.

**Goal:** Enable asset discovery and management via Tauri commands.

**Priority:** P0 - Critical (blocks asset functionality)

**Estimated Hours:** 12-18h

**Dependencies:** None (foundation epic)

## Technical Notes

### Asset Types
```rust
// src-tauri/src/types/mod.rs

#[derive(Serialize, Deserialize, specta::Type)]
pub struct IconInfo {
    pub name: String,
    pub path: PathBuf,
    pub width: Option<u32>,
    pub height: Option<u32>,
    pub category: Option<String>,
}

#[derive(Serialize, Deserialize, specta::Type)]
pub struct LogoInfo {
    pub name: String,
    pub path: PathBuf,
    pub variants: Vec<LogoVariant>,
}

#[derive(Serialize, Deserialize, specta::Type)]
pub struct LogoVariant {
    pub mode: String,  // "light", "dark", "default"
    pub path: PathBuf,
}
```

### Discovery Locations
Standard locations to scan:
- `src/assets/icons/`
- `public/icons/`
- `assets/icons/`
- `src/assets/logos/`
- `public/logos/`
