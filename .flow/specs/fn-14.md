# fn-14: Typography Editor

## Overview

Build the Typography Editor feature as specified: a styleguide view showing all HTML elements with editable properties and font management.

**Goal:** Enable visual editing of base typography (h1-h6, p, a, etc.) and font management.

**Priority:** P0/P1 - Core editor feature

**Estimated Hours:** 30-45h

**Dependencies:** fn-9 (typography write commands)

## Task Breakdown

### Phase 1: Styleguide View (P0)

**fn-14.1: Create Styleguide Component** (3-4h)
- Create `StyleguideView.tsx`
- Render all spec elements: h1-h6, p, a, ul, ol, li, blockquote, code, pre
- Load actual theme styles
- Responsive layout

**fn-14.2: Implement Element Selection** (2h)
- Click element to select
- Highlight selected element
- Show element type and selector
- Open properties panel

**fn-14.3: Connect to Properties Panel** (2h)
- Pass selected element to Properties
- Show typography properties
- Enable editing
- Wire to write commands

### Phase 2: Font Manager (P0)

**fn-14.4: Create Font Discovery Command** (2-3h)
- Scan project for @font-face rules
- Parse font files in assets
- Return font metadata
- Handle font loading errors

**fn-14.5: Build Font List Panel** (3-4h)
- Create `FontManager.tsx`
- Show installed fonts
- Font preview text
- Font weight/style variants

**fn-14.6: Implement Font Selection** (2-3h)
- Font picker dropdown
- Search fonts
- Recently used fonts
- System fonts fallback

### Phase 3: Properties Panel (P1)

**fn-14.7: Build Typography Properties Panel** (4-5h)
- Create `TypographyProperties.tsx`
- Font family picker
- Size (with token scale)
- Weight, style, line-height
- Color (token picker)

**fn-14.8: Add Token Scale Options** (3-4h)
- Show font-size token options
- spacing-* tokens for margins
- Preview scale changes
- Apply token or custom value

**fn-14.9: Add Color Token Picker** (1-2h)
- Show color tokens for text
- Preview color on element
- Apply token to color property

### Phase 4: Advanced Features (P1)

**fn-14.10: Implement Tag Conversion** (3-4h)
- Convert between semantic tags
- h1 → h2, p → blockquote
- Update CSS rules accordingly
- Preview before converting

**fn-14.11: Add Rust Font Parsing** (4-6h)
- Parse @font-face rules
- Extract font metadata
- Support woff, woff2, ttf
- Calculate font metrics

## Technical Notes

### Styleguide Elements (per spec)
```tsx
const STYLEGUIDE_ELEMENTS = [
  { tag: 'h1', label: 'Heading 1', sample: 'The quick brown fox' },
  { tag: 'h2', label: 'Heading 2', sample: 'The quick brown fox' },
  { tag: 'h3', label: 'Heading 3', sample: 'The quick brown fox' },
  { tag: 'h4', label: 'Heading 4', sample: 'The quick brown fox' },
  { tag: 'h5', label: 'Heading 5', sample: 'The quick brown fox' },
  { tag: 'h6', label: 'Heading 6', sample: 'The quick brown fox' },
  { tag: 'p', label: 'Paragraph', sample: 'Lorem ipsum...' },
  { tag: 'a', label: 'Link', sample: 'Click here' },
  { tag: 'ul', label: 'Unordered List', sample: ['Item 1', 'Item 2'] },
  { tag: 'ol', label: 'Ordered List', sample: ['First', 'Second'] },
  { tag: 'blockquote', label: 'Blockquote', sample: 'Famous quote...' },
  { tag: 'code', label: 'Code', sample: 'const x = 1;' },
  { tag: 'pre', label: 'Preformatted', sample: 'function() {...}' },
];
```

### Typography Write Format
```rust
// Target: @layer base in CSS
h1 {
  font-family: var(--font-heading);
  font-size: var(--font-size-4xl);
  font-weight: 700;
  line-height: 1.2;
  color: var(--color-text-primary);
}
```

### Font Metadata Structure
```typescript
interface FontInfo {
  family: string;
  weights: number[];
  styles: ('normal' | 'italic')[];
  source: 'local' | 'google' | 'custom';
  path?: string;
  previewUrl?: string;
}
```
