# fn-12: Search & Navigation

## Overview

Implement content search, keyboard navigation, and preferences. Currently search is non-functional and many shortcuts are missing.

**Goal:** Enable fast content discovery and power-user navigation.

**Priority:** P0/P1 - Critical for productivity

**Estimated Hours:** 40-60h

**Dependencies:** fn-9 (needs content to search), fn-11 (some features need git)

## Task Breakdown

### Phase 1: Search System (P0)

**fn-12.1: Create Search Index** (4-6h)
- Build searchable index of components, tokens, icons
- Use fuzzy-matcher for scoring
- Index on project load
- Update on file changes

**fn-12.2: Build Search Overlay UI** (4-6h)
- Create `SearchOverlay.tsx`
- Cmd+F triggers overlay
- Real-time filtering
- Category tabs (All, Components, Tokens, Icons)

**fn-12.3: Implement Search Navigation** (2-3h)
- Arrow key navigation through results
- Enter navigates to item
- Escape closes overlay
- Recent searches history

**fn-12.4: Wire Undo/Redo Shortcuts** (3-4h)
- Connect Cmd+Z to undo
- Connect Cmd+Shift+Z to redo
- Integrate with edit history
- Visual feedback

**fn-12.5: Fix Canvas Section Shortcuts** (2-3h)
- Audit current shortcut bindings
- Fix wrong targets (per review)

**fn-12.6: Build Native Menu Bar** (4-6h)

**fn-12.12: Implement Preference Persistence** (2-3h)

**fn-12.13: Add Key Shortcut Customization** (3-4h)

  components: SearchItem[];
  tokens: SearchItem[];
  icons: SearchItem[];
  pages: SearchItem[];
}

interface SearchItem {
  id: string;
  name: string;
  type: 'component' | 'token' | 'icon' | 'page';
  path: string;
  keywords: string[];
}
