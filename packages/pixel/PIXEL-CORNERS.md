# Pixel Corners

The current package-level corner documentation lives in [README.md](./README.md#pixel-corners).

This file remains as a stable pointer for older links. The old `xs/sm/md/lg/xl`
corner notes are obsolete; current authored CSS utilities are numeric
(`pixel-rounded-2`, `pixel-rounded-4`, `pixel-rounded-6`, ...), and runtime corner
composition uses `corner` helpers plus `px()`.
