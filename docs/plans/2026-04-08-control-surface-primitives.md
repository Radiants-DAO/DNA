# Control Surface Primitives

Style reference: Layout panel (gold-on-black terminal aesthetic, monospace, beveled/etched borders, section headers with rule lines).

## Continuous Value Controls

- [ ] **Knob** — Rotary encoder, continuous or stepped/detented. *(Ableton, Serum, modular synths)*
- [ ] **Fader** — Vertical slider with track + thumb. *(Mixing consoles, iOS volume)*
- [ ] **Slider** — Horizontal variant of fader. *(CSS property editors, EQ bands)*
- [ ] **XY Pad** — 2D parameter space with dot-grid background. *(Kaoss Pad, Serum mod matrix)*
- [ ] **Number Scrubber** — Drag-to-adjust inline value, the "invisible slider". *(Blender, Figma)*
- [ ] **Ribbon** — Horizontal continuous strip, no discrete thumb. *(Synth pitch strips, touch bars)*
- [ ] **Arc / Ring** — Circular progress/value, knob's read-heavy cousin. *(Ableton macro knobs, activity rings)*

## Discrete Selection Controls

- [ ] **Segmented Control** — Mutually exclusive mode selector with icon+label. *(Flex/Grid/Block/None row)*
- [ ] **Stepper** — Increment/decrement with editable center value. *(Blender spinners)*
- [ ] **Button Strip** — Row of icon buttons, radio or multi-select. *(DAW transport, align/justify rows)*
- [ ] **Toggle** — On/off switch, lamp, or latching button. *(Bypass buttons, DAW solo/mute)*
- [ ] **Chip / Tag** — Selectable label pills. *(VS Code filters, Ableton clip colors)*
- [ ] **Matrix Grid** — On/off grid, step sequencer / drum machine style. *(808 pattern editor, Bitwig)*
- [ ] **Radial Menu** — Pie-style contextual selector. *(Blender pie menus, Maya marking menus)*

## Readout / Feedback Controls

- [ ] **Meter** — Segmented or continuous read-only bar. *(VU meters, level meters, CPU monitors)*
- [ ] **Waveform** — Mini waveform thumbnail / oscilloscope. *(DAW waveform views, audio editors)*
- [ ] **Spectrum** — Frequency or data distribution readout. *(EQ spectrum analyzers, FFT displays)*
- [ ] **Sparkline** — Tiny inline graph, trend-at-a-glance. *(Grafana, dev tools performance)*
- [ ] **LED Array** — Row/grid of status lamps. *(Hardware status indicators, MIDI controllers)*

## Editor / Compound Controls

- [ ] **Curve Editor** — Bezier graph with draggable control points. *(After Effects, Houdini, ADSR envelopes)*
- [ ] **Crossfader** — Horizontal blend between two states. *(DJ mixers, A/B blend controls)*
- [ ] **Color Swatch** — Trigger for picker; the swatch itself is a control. *(Every design tool ever)*
- [ ] **Minimap** — Tiny overview of a larger space. *(VS Code, Unreal viewport)*
- [ ] **Node Socket** — Connection point for patch/flow UIs. *(Houdini, Blender shader editor)*

## Structural / Container Controls

- [ ] **Collapsible Section** — Panel header with rule line + collapse state. *(SIZE/DISPLAY/POSITION headers)*
- [ ] **Dropdown** — Value-from-list, compact. *(Unit selectors: REM, PX)*
- [ ] **Scrub Timeline** — Horizontal time/frame scrubber with playhead. *(DAW timeline, video editors)*
