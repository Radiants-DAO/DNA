# Docs

Current documentation lives here.

* `production-readiness-checklist.md` is the live backlog.

* `plans/` contains current, unimplemented execution plans.

* `research/` contains reusable prompt docs and any active research workspaces.

* `ops/` contains current operational workflows, checklists, and append-only logs.

* `solutions/` contains durable implementation notes and decisions.

* Completed research runs, historical plans, brainstorms, reports, and old prompt packs live in `../archive/`.

* Future concepts and spike artifacts live in `../ideas/`.
