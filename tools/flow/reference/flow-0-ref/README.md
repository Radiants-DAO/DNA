# RadFlow Tauri

Design System Manager for LLM CLI tools (Claude Code, Cursor, etc.).

## Status

Active Development - Research & Planning Phase

## Quick Start

```bash
pnpm tauri dev      # Development
pnpm tauri build    # Production build
cargo test          # Rust tests
```
