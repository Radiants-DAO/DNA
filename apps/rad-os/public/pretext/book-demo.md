# Small Systems, Big Type

Design systems get more useful when their text primitives can paginate, reflow, and serialize without binding themselves to a single editor implementation.

## Chapter One

The book primitive reuses the shared paginator, so the same normalized markdown can become a pamphlet, a manifesto, or a longer booklet excerpt.

## Chapter Two

When a text surface can move between markdown, preview, and exported files without losing structure, the surrounding app gets simpler too.
