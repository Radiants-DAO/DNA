# Scratchpad

A block-based editor inside RadOS, powered by **BlockNote** + **RDNA** design system components. Type `/` to open the command menu.

## Typography

# Heading 1
## Heading 2
### Heading 3

Body text in *Mondwest*. Headings render in *Joystix*.

## Inline Formatting

**Bold**, *italic*, ~~strikethrough~~, and `inline code`.

## RDNA Components

All 39 RDNA design system components are available via the `/` slash menu. Type `/alert`, `/card`, `/button`, `/tabs`, etc. to insert them.

### Feedback
- `/alert` — Contextual message (info, success, warning, error variants)
- `/badge` — Status indicator label
- `/toast` — Notification placeholder
- `/tooltip` — Hover context placeholder

### Layout
- `/card` — Container with pixel-corner surface (default, inverted, raised)
- `/separator` — Visual divider
- `/collapsible` — Expandable content section
- `/tabs` — Tabbed navigation
- `/pattern` — Decorative background pattern
- `/scrollarea` — Scrollable container
- `/appwindow` — Window placeholder

### Form Controls
- `/input` — Text input field
- `/numberfield` — Numeric input with increment/decrement
- `/select` — Dropdown selector
- `/combobox` — Searchable dropdown
- `/checkbox` — Check toggle
- `/switch` — On/off toggle
- `/slider` — Range input
- `/toggle` — Press toggle button
- `/togglegroup` — Segmented toggle group
- `/inputset` — Field group wrapper

### Actions
- `/button` — CTA button (filled, outline, text modes)
- `/toolbar` — Action bar with buttons

### Overlays
- `/dialog` — Modal dialog placeholder
- `/alertdialog` — Confirmation dialog placeholder
- `/sheet` — Side panel placeholder
- `/drawer` — Bottom drawer placeholder
- `/popover` — Popover placeholder
- `/dropdownmenu` — Dropdown menu placeholder
- `/contextmenu` — Right-click menu placeholder
- `/previewcard` — Hover preview placeholder

### Navigation
- `/breadcrumbs` — Breadcrumb trail
- `/navigationmenu` — Nav menu placeholder
- `/menubar` — Menu bar placeholder

### Data Display
- `/meter` — Progress/value meter
- `/countdowntimer` — Live countdown display
- `/avatar` — User identity display
- `/icon` — SVG icon
- `/spinner` — Loading indicator

---

## Slash Commands

Type `/` anywhere to insert any block type. Search by name, alias, or category. All 39 RDNA components are available alongside standard blocks (headings, lists, tables, code, images).
